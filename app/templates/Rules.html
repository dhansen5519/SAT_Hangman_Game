Rules of the Game

Starting the Game

  • A word is selected at random
  • The selected word is displayed as a series of blank spaces. Each space is a letter in the word.
  • The word definition is displayed

Making Guesses

  • Select a single letter to guess. If you are correct one or more of the blanks will fill in.
  • If you select a wrong letter one piece of the hangman image will be added.

Finishing the Game

  • You win the game when you have guessed every letter of the word correctly before the hangman image fills in completely.
  • A message will display when you have won or lost the game. Good Luck!
